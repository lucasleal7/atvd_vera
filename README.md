Análise de Ações de Empresas de Tecnologia
Este projeto realiza uma análise das ações de quatro grandes empresas de tecnologia: Apple (AAPL), Google (GOOG), Microsoft (MSFT) e Amazon (AMZN). Utilizando dados financeiros históricos, o notebook realiza a coleta, limpeza e visualização dos dados, bem como análises exploratórias e previsões para entender o comportamento das ações ao longo do tempo.

Tecnologias Utilizadas
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
Seaborn
yFinance
Scikit-learn (para modelagem e previsões)

Estrutura do Projeto
acoes.ipynb: Notebook principal contendo a análise das ações.
requirements.txt: Arquivo com as dependências do projeto.
Exemplo de Análise
O notebook realiza as seguintes análises:

Coleta de Dados: Utiliza a biblioteca yFinance para coletar dados históricos das ações.
Limpeza de Dados: Organiza e limpa os dados para análise.
Visualizações: Gera gráficos para visualizar o comportamento das ações ao longo do tempo.
Análises Exploratórias: Explora estatísticas descritivas e correlações entre as ações das empresas.
Modelagem e Previsões: Utiliza técnicas de aprendizado de máquina para prever o comportamento futuro das ações.
